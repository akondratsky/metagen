# #copy

The **#copy** node tells to the Metagen that the current file should not be processed by Handlebars. Such files will be just copied.

<Tabs>
<TabItem value="input" label="Input">

```txt title="{#copy}template.hbs"
{{value}}
```

</TabItem>
<TabItem value="payload" label="Payload">

```json
{ "value": "42" }
```

</TabItem>
<TabItem value="output" label="Output">

```txt title="template.hbs"
{{value}}
```

</TabItem>
</Tabs>
